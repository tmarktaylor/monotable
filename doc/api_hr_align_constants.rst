Vertical Alignment Constants
----------------------------

For convenience these names are copied to the monotable namespace by
__init__.py.  They are unique int enumeration values.
They are accessible as::

   monotable.TOP
   montable.CENTER_TOP
   montable.CENTER_BOTTOM
   montable.BOTTOM
